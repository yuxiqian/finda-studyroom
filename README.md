# finda-studyroom

## 能做什么？

- [x] 从 electsys.sjtu.edu.cn 抓取到指定学期 XML、CSV 或 HTML 等格式的上课教室数据

- [ ] 分析整栋楼的教室数据，得出该学期指定天的教室占用情况

- [ ] 将得到的数据可视化

- [ ] 智能提供自习教室选择推荐
